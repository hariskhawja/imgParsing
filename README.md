# imgParsing

## Description
Python module built upon Pillow for basic RGB image parsing with a bonus image file type conversion function.

## Installation

#### Clone Repository
```
git clone https://github.com/hariskhawja/imgParsing.git
```

#### Import
```
import imgParsing
```
